# Dorm Dash – Triton Eats

A UCSD-specific food delivery web app, styled after DoorDash, built with Next.js 14, TypeScript, and Tailwind CSS.

## Student Order Flow

1. **Home** (`/`) — Browse UCSD dining halls (64 Degrees, Pines, Sixth Market, etc.)
2. **Order Queue** (`/order-queue`) — Verify your order: last 4 of student ID, target pickup time, and Triton2Go container confirmation (screenshot this page!)
3. **Location** (`/location`) — Pick a campus pickup spot or opt for dorm room delivery
4. **Matching** (`/matching`) — Animated Dasher matching screen
5. **Live Delivery** (`/delivery`) — Real-time order status tracker + chat with your Dasher
6. **Complete** (`/complete`) — Push notification sent to Dasher, order summary, rate your Dasher

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).
